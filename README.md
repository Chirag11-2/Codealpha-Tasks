# CodeAlpha Software Development Internship

This repository contains the projects completed during my Software Development Internship at CodeAlpha.

## Projects

### Task 1 - Audio Player
A responsive web-based audio player built using HTML, CSS, and JavaScript.

Features:
- Play/Pause Music
- Next & Previous Song
- Progress Bar
- Volume Control
- Responsive UI

### Task 2 - Calculator
A modern calculator developed using HTML, CSS, and JavaScript.

Features:
- Basic Arithmetic Operations
- Keyboard Support
- Responsive Design
- Error Handling

### Task 3 - Image Gallery
An interactive image gallery built using HTML, CSS, and JavaScript.

Features:
- Category Filter
- Lightbox Preview
- Previous & Next Navigation
- Responsive Layout

## Technologies Used
